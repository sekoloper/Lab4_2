# Lab4_2
Программа представляет из себя вторую часть лабораторной работы под номером 4 по дисциплине «Язык программирования C#».

## Требования
* Windows 10 (или выше)
* Visual Studio

## Установка и запуск программы
1. Клонируйте репозиторий в Visual Studio
2. Соберите решение (Ctrl+Shift+B)
3. Запустите (Ctrl+F5)

## Руководство по использованию
1. Введите номер задачи.
2. Следуйте дальнейшим инструкциям внутри задачи.
3. После выполнения задачи введите номер новой или 0 для выхода из программы.

## Структура проекта
Lab4_2
* Program.cs
* InputValidation.cs
* QuadraticEquation.cs

Program.cs – основной файл, содержащий класс Program и метод Main. В нём проводится проверка работоспособности класса QuadraticEquation.

InputValidation.cs – файл с функцией корректного ввода числа типа double.

QuadraticEquation.cs – файл, в котором содержится класс QuadraticEquation.

## Задачи

##Задание 6.

Формулировка: 

Название класса: QuadraticEquation 
Поля: double a, double b, double c (коэффициенты)
Методы: Вычисление корней квадратного уравнения. 
Результат должен быть массивом величин типа double 
(в нем от 0 до 2-х элементов, в зависимости от количества корней).

Алгоритм: 

Был создан класс с соответствующими полями _a, _b, _c,
Метод Solve для вычисления корней.

Результат:

1*x^2 + 3*x + -4 = 0
Корни уравнения:
1
-4

##Задание 7.

Формулировка: 

Унарные операции:
++ увеличивает коэффициенты уравнения на 1;
-- уменьшает коэффициенты уравнения на 1.
Операции приведения типа:
double (неявная) – результатом является дискриминант уравнения.
bool (явная) – результатом является true, если корни существуют и falseв
противном случае;
Бинарные операции
== QuadraticEquationt1, QuadraticEquationt2 — уравнения равны, если
равны их коэффициенты;
!= QuadraticEquation1, QuadraticEquation2 — уравнения не равны, если не
равны их коэффициенты.

Алгоритм: 

Созданы соответствующие унарные, бинарные операции и операции приведения типа.

Результат:

1*x^2 + 3*x + -4 = 0
2*x^2 + 4*x + -3 = 0
1*x^2 + 3*x + -4 = 0
Есть корни? True
Дискриминант: 25
